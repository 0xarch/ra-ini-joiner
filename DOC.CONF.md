# 配置

*   `InputRoot`：项目根目录
*   `OutputPath`：输出文件路径
*   `Macro.Root`：宏文件目录
*   `Macro.IgnoreKeys`：要被忽略的键
*   `Macro.ExplicitKeys`：显式指定的键
*   `Macro.Overrides`：映射表
*   `Macro.DefaultFileSuffix`：默认文件后缀
*   `Macro.AutoDetect`：自动检测参数宏
*   `IgnoreFiles`：忽略的文件
*   `ExplicitRequiredFiles`：显式要求索引的文件
*   `IsDetailedConsole`：显示更多输出
